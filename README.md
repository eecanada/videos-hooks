# Description
 
- React Video is a react application that allows users to search for videos using Youtube's API.
 
# Technologies Used
 
- JavaScript
- CSS3
- HTML5
- Semantic UI
- React.js
 
# Getting Started
 
- The user loads single page application and it will always load with "james bond no time to die" as the search query.
 
- The user can search for another video if they please and when they do a request the first 5 videos will load.
 
- A user can click any of the videos on the sidebar and it will become the main video on the page to be viewed.
